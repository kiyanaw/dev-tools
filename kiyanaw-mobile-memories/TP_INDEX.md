# Template: Index Files (*_INDEX)

## Structure

```markdown
# [NAME]_INDEX - [Purpose]

## Quick Lookup

| [Key] | [Value] | [Notes] |
|-------|---------|---------|
| item  | target  | brief   |

## Categories (if needed)

### Category A
- Item → Target

### Category B
- Item → Target
```

## Rules

- **Lookup table format** - Fast scanning
- **No prose** - Tables and lists only
- **Keep updated** - Stale indexes cause drift
- **Link to details** - Index points to full docs

## Checklist

- [ ] Table format for quick lookup?
- [ ] No unnecessary prose?
- [ ] All entries current?
- [ ] Links to detailed docs?
